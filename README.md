# testgame

Простая онлайн-игра «Крестики-нолики» для 2 игроков.

## Как играть

1. Открой страницу проекта на GitHub Pages.
2. Игрок 1 нажимает **«Создать игру»** и отправляет ссылку другу.
3. Игрок 2 открывает ссылку — подключение автоматически.
4. Игра идёт в реальном времени.

## Технологии

- HTML/CSS/JS
- PeerJS (WebRTC через публичный signaling-сервер PeerJS)

## Важно

Это P2P-игра без серверной логики репозитория. Если PeerJS облачный сервер временно недоступен, соединение может не установиться.
